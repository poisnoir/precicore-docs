# Input

The Input layer handles all operator control signals entering the PreciCore system. Raw input is captured, processed through Purifier for tremor filtering, and forwarded to the control system over Spine.

## Supported Input Devices

| Device | Description | Status |
|--------|-------------|--------|
| Xbox Controller | Baseline joystick input | ✅ Active |
| iPhone IMU | Wrist motion-based control via IMU | ✅ Active |
| Keyboard | Development and testing input | ✅ Active |
