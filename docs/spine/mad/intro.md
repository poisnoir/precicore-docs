# MAD

MAD is a fast binary serializer used within the Spine framework for compact, high-performance message encoding. Designed for low-latency communication between PreciCore modules over UDP.

## Why MAD?

JSON and other text-based formats introduce unnecessary overhead for real-time robotics systems. MAD encodes messages into compact binary format, minimizing packet size and serialization time.
